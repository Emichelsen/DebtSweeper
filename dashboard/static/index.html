About the Graph

The DebtSweeper Knowledge Graph visualizes your code structure and technical debt:

  • Nodes: Represent code modules, classes, and functions
  • Node size: Indicates complexity (larger = more complex)
  • Node color: Shows test coverage (green = good, red = poor)
  • Edges: Show dependencies between code components

Interact with the graph by clicking nodes to see details, dragging to reposition, and zooming to explore.
